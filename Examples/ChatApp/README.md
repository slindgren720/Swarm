# Chat App Example

A complete SwiftUI chat application showcasing SwiftAgentsUI components.

## Overview

This example demonstrates:
- Building a chat interface with `ChatView`
- Integrating agents with SwiftUI
- Displaying tool results inline
- Managing conversation state

## Requirements

- iOS 26.0+ / macOS 26.0+
- Swift 6.2+
- Xcode 26.0+
- SwiftAgentsUI framework
