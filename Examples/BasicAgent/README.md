# Basic Agent Example

A minimal example demonstrating how to create and run a simple agent with SwiftAgents.

## Overview

This example shows:
- Creating an agent with `AgentProtocol`
- Configuring agent settings
- Running a basic conversation loop
- Handling agent responses

## Requirements

- iOS 26.0+ / macOS 26.0+
- Swift 6.2+
- Xcode 26.0+

## Usage

```swift
import SwiftAgents

// Example code will be added in future implementation
```

## What You'll Learn

1. **Agent Configuration** - Setting up an agent with custom parameters
2. **Tool Registration** - Adding tools to extend agent capabilities
3. **Conversation Flow** - Managing input/output in a chat loop
4. **Error Handling** - Gracefully handling agent failures
